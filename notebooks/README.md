# JupyterLite dans votre navigateur.

Il suffit de déposer vos dossiers, notebooks et fichiers dans ce répertoire pour commencer à travailler.
A la différerence de jupyerlab, ici Python tourne **DANS** dans votre navigateur. Tout est déjà préinstallé, vous n'avez rien à configurer. 

⚠️ **Mais attention** : 

- ‼️ *si vous travaillez depuis une machine de la Platerforme Pédagogique d'Ingénierie (PPI)* : toutes les données que vous déposerez ici n'existeront que dans le navigateur de votre session active, et ne seront pas partagées avec d'autres sessions et d'autres ordinateurs. Par ailleurs, toutes les données déposées dans ce navigateur seront perdues à la déconnexion. Pensez donc à exporter vos notebooks pour les stocker définitivement sur votre propre ordinateur à la fin de la séance de TP ;
- ‼️ *si vous utilisez votre propre ordinateur* : toutes les données que vous déposerez ici n'existeront que dans **votre** navigateur et ne seront pas partagées avec d'autres sessions et d'autres ordinateurs. Par ailleurs, toutes vos données seront perdues si vous videz cache et/ou cookies pour ce site. Exportez donc régulièrement vos notebooks pour les stocker définitivement dans vos propres documents.
