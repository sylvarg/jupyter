# JupyterLite dans votre navigateur.

Il suffit de déposer vos dossiers, notebooks et fichiers dans ce répertoire pour commencer à travailler.
A la différerence de jupyerlab, ici Python tourne **DANS** dans votre navigateur. Tout est déjà préinstallé, vous n'avez rien à configurer. 

⚠️ **Mais attention** : toutes les données que vous déposerez ici n'existeront que dans **votre** navigateur et ne seront pas partagées avec d'autres sessions et d'autres ordinateurs. Par ailleurs, toutes vos données seront perdues si vous videz cache et/ou cookies pour ce site. Exportez donc régulièrement vos notebooks pour les stocker définitivement dans vos propres documents.
